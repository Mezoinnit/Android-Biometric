# Project Context: BiometricFeature

## Project Overview

**BiometricFeature** is an Android application project built using Kotlin and Jetpack Compose.
Currently, it appears to be in the initial setup phase, containing the boilerplate code generated by Android Studio for an "Empty Compose Activity". Despite the name, there are no specific biometric dependencies or implementations present yet.

## Tech Stack

*   **Language:** Kotlin
*   **UI Framework:** Jetpack Compose (Material3)
*   **Build System:** Gradle (Kotlin DSL)
*   **Minimum SDK:** 30 (Android 11)
*   **Target SDK:** 35 (Android 15)
*   **Compile SDK:** 35

## Key Files & Directories

*   `app/src/main/java/com/mezo/biometric/MainActivity.kt`: The main entry point of the application. Currently contains a basic `Scaffold` setup.
*   `app/build.gradle.kts`: Module-level build configuration.
*   `gradle/libs.versions.toml`: Version catalog defining dependencies and plugins.
*   `build.gradle.kts`: Root-level build configuration.

## Features

*   **Biometric Vault:** Encrypt and Decrypt text using Biometric-bound keys.
*   **Secure Storage:** Uses Android Keystore and SharedPreferences (Ciphertext/IV).
*   **Quality UI:** Material 3 Design, Password Toggle, Feedback (Snackbar).
*   **Testing:** Unit tests for Data layer (`VaultRepository`).

## Building and Running

### Command Line

*   **Build Debug APK:**
    ```bash
    ./gradlew assembleDebug
    ```
*   **Run Unit Tests:**
    ```bash
    ./gradlew testDebugUnitTest
    ```
*   **Run Instrumented Tests:**
    ```bash
    ./gradlew connectedAndroidTest
    ```

### Android Studio

This project is designed to be opened and run within Android Studio. Sync the project with Gradle files upon opening.
